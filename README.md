
## 目录结构
```shell
├─api                               // 接口封装
├─assets                            // 静态资源
├─components                        // 可复用的组件
├─config                            // axios封装
├─filters                           // 全局filters
├─pages                             // 组件
│  ├─acci-logging                   // 次要页面
│  ├─acci-systemSetting             // 次要页面
│  ├─errorPages                     // 错误
│  ├─home                           // 主页
│  │  ├─admin
│  │  └─manager
│  ├─layout                         // 布局页面
│  ├─login                          // 登陆
│  └─residents                      // 居民列表
│      ├─admin
│      └─manager
├─router                            // 路由
├─store                             // vuex
├─styles                            // 全局样式
├─utils                             // 工具
├─App.vue                           // 项目入口
└─main.js                           // 项目入口
```
